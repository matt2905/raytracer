Raytracer
=========

Réalisation d'un programme de calcul d’image de synthese
grâce a la minilibix : une librairie graphique simplifié de la X11.

Utilisation
===========

Compilation avec Make <br/>

Excution via la commande: ./rt <code><scene></code><br/>
